Getting Started

To use the Leap Year Checker program, follow these steps:

1. Make sure you have Python installed on your system.
2. Clone or download the repository.

Prerequisites
Python 3.x

Usage

1. Open a terminal or command prompt.
2. Navigate to the directory where the code file is located.
3. Run the following command to execute the script: leapYearChecker.py

The program will ask you to enter a year as input. It will then calculate whether the year is a leap year or not based 
on the specified criteria. The result will be printed.

Example

Which year do you want to check: 2020
We have a leap year

Which year do you want to check: 1900
Nope! Not a leap year

Which year do you want to check: 2000
Aha, we have a leap year

Feel free to modify the code or incorporate it into your own projects. Enjoy checking leap years!
